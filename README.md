A WordPress plugin that helps you automatically extract long-tail SEO keywords from posts and insert them into Rank Math and Yoast SEO. Developed by: Mona Jalal.

== Description ==

**Long Tail Keywords SEO Booster** is a smart plugin that analyzes your post content and title to automatically generate **long-tail keywords**, and inserts them into your SEO plugin fields:

- ✅ Rank Math SEO
- ✅ Yoast SEO

**Developed by: Mona Galal** ❤️

**Features:**

- Automatically generates long-tail keywords from post title and content.
- Adds them as Focus Keywords in Rank Math & Yoast.
- Manual keyword generation button in each post editor.
- Bulk generation for all posts in one click.
- Generate keywords for "Today’s Posts Only" (Published, Scheduled, Draft).
- Daily auto-run with WordPress Cron enabled.

== How to Use ==

1. Activate the plugin from the WordPress dashboard.
2. Go to: Tools > Long Tail Keywords - Bulk.
3. Use one of the available buttons:
   - “Generate for All Posts”
   - “Generate for Today’s Posts Only”
4. You will also find a button in the sidebar of each post editor.

== Frequently Asked Questions ==

= Does it work with custom post types? =
Currently, only standard blog posts. But you can expand it easily to support custom post types like WooCommerce products.

= Does it use AI or external APIs? =
No, the plugin uses internal logic to extract high-frequency long-tail phrases from the content.

= Can it run automatically every day? =
Yes. The plugin uses WP-Cron to generate keywords daily at midnight (server time).

== Installation ==

1. Download the plugin ZIP file.
2. Go to WordPress Admin → Plugins → Add New → Upload Plugin.
3. Upload and activate the plugin.
4. Use the options under “Tools > Long Tail Keywords - Bulk”.
